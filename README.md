## 📖 About this Dashboard  

Dashboard_Producao_PowerBI is an interactive production dashboard built using Power BI. It organizes and presents production data clearly. The data is cleaned and prepared using Power Query. Key business indicators are created with DAX formulas. This tool helps users track production performance and make informed decisions using reliable data.  

You do not need coding skills to use it. The dashboard works on any Windows computer with Power BI Desktop installed.  

Main features include:  
- Data cleaning and shaping with Power Query  
- Custom indicators and KPIs calculated by DAX  
- Easy-to-understand visuals and charts  
- Filters and interactive elements for quick analysis  
- Supports production data from Excel or other sources  

This dashboard fits users who want to monitor manufacturing or production data efficiently.

## 🛠 Troubleshooting Tips  

- If the dashboard does not open, confirm Power BI Desktop is installed correctly.  
- Make sure your production data files are not open in other programs when refreshing.  
- If data does not update, check that file paths and database connections are correct in Power BI Desktop.  
- For missing visuals or errors, try updating Power BI Desktop to the latest version.
