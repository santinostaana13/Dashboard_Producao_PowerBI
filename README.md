# 📊 Dashboard de Produção - Power BI
<img width="1076" height="617" alt="image" src="https://github.com/user-attachments/assets/ad91208f-4502-49bc-8ab0-1aa1751cc17d" />

Projeto desenvolvido durante o **Intensivão de Power BI da Hashtag Treinamentos**, com foco em tratamento de dados, criação de medidas em DAX e construção de dashboard interativo para análise de produção industrial.

---

##  Objetivo do Projeto

Construir um dashboard completo para análise de desempenho produtivo, permitindo visualizar:

- Horas produtivas
- Horas paradas
- Disponibilidade (%)
- Quantidade produzida
- Quantidade rejeitada
- Qualidade (%)
- Análise mensal da produção
- Filtros por operador e mês

---

##  Ferramentas Utilizadas

- Power BI
- Power Query (tratamento de dados)
- DAX (criação de medidas)
- Excel (base de dados)

---

##  Tratamento de Dados (Power Query)

Durante a etapa de transformação foram realizados:

- Correção de tipos de dados (número, data e hora)
- Remoção de linhas em branco
- Separação de colunas de Data e Hora
- Cálculo da duração total em horas
- Organização e padronização da base

---

##  Medidas Criadas em DAX

Algumas das principais medidas desenvolvidas:

- `Quantidade Produzida`
- `Quantidade Rejeitada`
- `Qualidade (%)`
- `Horas Produtivas`
- `Horas Paradas`
- `Disponibilidade (%)`

Utilização das funções:

- `SUM`
- `CALCULATE`
- Operações condicionais com filtros
- Cálculo de indicadores percentuais

---

##  Visualizações Criadas

- Cartões de indicadores (KPIs)
- Gráfico de Área (análise mensal)
- Gráficos de Indicador (velocímetro)
- Segmentação de dados (filtros suspensos)
- Layout personalizado com ícones e identidade visual
